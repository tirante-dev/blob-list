# blob-list generated data

This branch is machine-published by the `Publish Generated Data`
workflow. Do not edit it by hand and do not open pull requests
against it; every publish force-replaces the branch contents.

- `artifacts/` — generated registry artifacts for consumers.
- `data/chainlist/snapshot.json` — the pinned Chainlist snapshot
  the artifacts were generated from.

Stable URLs:

- `https://raw.githubusercontent.com/tirante-dev/blob-list/generated-data/artifacts/registry.json`
- `https://raw.githubusercontent.com/tirante-dev/blob-list/generated-data/artifacts/by-chain/{submission_chain}.json`

Versioned release assets with checksums are published at
<https://github.com/tirante-dev/blob-list/releases>.
